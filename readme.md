## Scissors and Glue for p5js

This project pairs with two arduino libraries scissors and glue.

Together these allow for easy assembly and parsing of simple delimited string based messages.

### Get Scissors and Glue for Arduino:

Both libraries are in the Arduino library manager.  Search for *scissors* OR *glue* OR *hex705* (author)
